Version 0.20.1 Released

The initial release of dcemriS4 on CRAN is now available. This package contains all the basic functionality of dcemri, plus an S4-class structure for both NIfTI and Analyze data formats along with streamlined graphical functions for medical imaging data. The source code is available for download (linux), along with appropriate versions for Mac OSX and Windows.
